# Working?
This is a feature in games that is called a **B**oss key. See Wikipedia article: (https://en.wikipedia.org/wiki/Boss_key).

## What is the correct boss key?
Press Ctrl+B

On consoles, you should press LT,RT,L and R at the same time.
